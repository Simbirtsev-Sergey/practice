# TechStore — каталог смартфонов

Учебный проект интернет-магазина (HTML, CSS, JavaScript) для практической
работы № 4 по дисциплине «Клиент-серверное программирование».

Студент: Юрков С. Д., группа БИВТ-24-1.

## Возможности

- каталог товаров с карточками;
- корзина: добавление, удаление товара, очистка, оформление заказа;
- **сохранение корзины в LocalStorage** — содержимое корзины восстанавливается
  после перезагрузки страницы (ветка `feature/LocalStorage`).

## Запуск

Открыть `index.html` в браузере (или запустить любой статический сервер,
например `npx serve`).

## Структура

```
pr-4-store/
├── index.html      # каталог и корзина
├── styles.css      # оформление
├── script.js       # логика корзины + LocalStorage
└── assets/         # изображения товаров (SVG)
```
